# Project: Bullet Manager with ArrayList

## Overview
Create a dynamic bullet management system where bullets can be spawned and removed.

## Requirements
1. ArrayList to store active bullets (positions, velocities)
2. spawn() method to add bullets
3. update() method to move bullets
4. removeOffscreen() method to clean up
5. Display bullet count and positions

## Grading: 100 points
- Bullet storage: 20 pts
- Spawn method: 20 pts
- Update method: 30 pts
- Remove method: 30 pts
