# Project: Maze Generator

## Overview
Create a maze generator that uses a 2D array to represent walls and paths.

## Requirements
1. 2D array to store maze (0=path, 1=wall)
2. Generate random maze layout
3. Ensure path from start to end exists
4. Display maze in ASCII art
5. Find and mark path (optional)

## Grading: 100 points
- Grid setup: 20 pts
- Maze generation: 30 pts
- Validation: 20 pts
- Display: 30 pts
