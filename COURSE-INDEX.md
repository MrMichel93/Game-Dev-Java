## üéØ Learning Path

### Beginner Track (Start Here!)
1. Complete [SETUP.md](SETUP.md) - Set up your environment
2. Read [Unit 1 README](Unit01-Primitive-Types/README.md)
3. Study [Lesson 1: Variables and Game State](Unit01-Primitive-Types/lessons/Lesson1-Variables-GameState.md)
4. Run [ScoreTracker.java](Unit01-Primitive-Types/code-examples/ScoreTracker.java)
5. Complete all three Unit 1 practice problems
6. Build the [Stat Display Panel project](Unit01-Primitive-Types/projects/StatDisplayPanel.md)
7. Move to Unit 2

### Intermediate Track (Units 2-7)
Complete Units 2-7 in sequence to build strong foundations in:
- Object usage and method calls
- Conditional logic and decision making
- Loops and iteration patterns
- Class design and encapsulation
- Arrays and ArrayLists

### Advanced Track (Units 8-10)
Master advanced topics:
- 2D arrays for grid-based systems
- Inheritance hierarchies and polymorphism
- Recursive algorithms
