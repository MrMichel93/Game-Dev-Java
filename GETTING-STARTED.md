# Getting Started with Java Game Development

Welcome to the Java Game Development course! This guide will help you get started quickly.

## üöÄ Quick Start (5 Minutes)

### Step 1: Check Prerequisites
- Java installed? Run `java -version` in terminal
- If not installed, see [SETUP.md](SETUP.md)

### Step 2: Choose Your First Lesson
**Recommended starting point:** [Unit 1, Lesson 1: Variables and Game State](Unit01-Primitive-Types/lessons/Lesson1-Variables-GameState.md)

### Step 3: Run Your First Example

```bash
cd Unit01-Primitive-Types/code-examples
javac ScoreTracker.java
java ScoreTracker
```

You should see a score tracking simulation!

### Step 4: Try Your First Problem
Open `Unit01-Primitive-Types/practice-problems/Problem1_HealthSystem.java` and implement the health system methods.

## üÜò Need Help?

### Common Issues:

**"Java not found"**
- Install JDK (see SETUP.md)
- Add to PATH environment variable

**"Class not found"**
- Make sure you're in the right directory
- Check spelling of class name

**"Cannot find symbol"**
- Check variable/method spelling
- Verify imports if using external libraries
