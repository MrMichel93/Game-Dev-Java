# Project: Level System with Arrays

## Overview
Create a level system that stores and manages game level data using arrays.

## Requirements
1. Array of level names
2. Array of level difficulties (1-5)
3. Array of high scores per level
4. Methods to: find hardest level, calculate total score, check completion

## Grading: 100 points
- Level data arrays: 25 pts
- Search/find methods: 25 pts
- Calculation methods: 25 pts
- Display/formatting: 25 pts
